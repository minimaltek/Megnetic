# Magnetic — 要件定義書

## 概要

白背景に黒いメタボールがうねうね動く、音とタッチに反応するビジュアルアプリ。  
シンプルで美しい磁性流体ライクなインタラクションをiPhone / iPadで提供する。

---

## プラットフォーム

| 項目 | 内容 |
|------|------|
| 対応デバイス | iPhone / iPad（Universal） |
| 最小OSバージョン | iOS 16.0以上 |
| 向き | Portrait / Landscape 両対応 |

---

## コア機能

### 1. メタボールレンダリング

- Metal Compute Shader でスカラーフィールドをGPU計算
- Marching Squares でアウトライン抽出 → 滑らかなメッシュ描画
- 白背景 × 黒メタボール
- 光沢ハイライト（球体感のある反射表現）
- ボール数：デフォルト8〜12個、設定で変更可能

### 2. マイク入力反応

- `AVAudioEngine` でリアルタイムマイク入力
- `vDSP` によるFFT（高速フーリエ変換）で周波数解析
- 周波数帯域ごとにメタボールの挙動を変化させる
  - 低音（〜200Hz）→ 全体が大きく膨張・収縮
  - 中音（200Hz〜2kHz）→ 軌道速度の変化
  - 高音（2kHz〜）→ 細かい振動・震え
- iPhone本体スピーカーから鳴らした音も拾える（内蔵マイク利用）

### 3. タッチインタラクション

- タッチ位置に引力が発生し、メタボールが引き寄せられる
- マルチタッチ対応（複数指で複数の引力点を同時生成）
- 指を離すとゆっくり元の軌道に戻る（バネ的な復元力）
- 長押しで反発力に切り替え（押しのけモード）

| 役割 | 技術 |
|------|------|
| UI | SwiftUI |
| レンダリング | Metal + MetalKit |
| 音声入力 | AVAudioEngine + vDSP (FFT) |
| 物理シミュレーション | カスタム実装（簡易SPH） |
| データ永続化 | UserDefaults（設定値のみ） |

## 画面構成

### メイン画面
- フルスクリーンのMetalビュー（メタボール描画）
- 上部に薄くメニューボタン（半透明）
- タップで設定パネルをトグル

### 設定パネル（ボトムシート）
- マイク感度スライダー
- ボール数スライダー（4〜20個）
- アニメーション速度スライダー
- Pro機能へのアップグレードボタン（無料ユーザーのみ）

### マイク許可フロー
- 初回起動時にマイク使用許可を説明するオンボーディング画面
- 許可拒否の場合はタッチのみのモードで動作

## 開発フェーズ

### Phase 1 — Metal メタボール基本描画
- MetalView の SwiftUI 組み込み
- Compute Shader でスカラーフィールド計算
- Marching Squares によるアウトライン描画
- 自動アニメーション（軌道運動）

### Phase 2 — タッチ引力
- UITouch → Metal バッファへの座標受け渡し
- 引力 / 反発力の物理計算
- マルチタッチ対応

### Phase 3 — マイク + FFT 連携
- AVAudioEngine セットアップ
- vDSP FFT 実装
- 周波数バンド → メタボールパラメータのマッピング

### Phase 4 — UI + 設定
- 設定パネル（ボトムシート）
- オンボーディング
- StoreKit 2 IAP 実装

### Phase 5 — App Store 申請
- TestFlight ベータテスト
- スクリーンショット・プレビュー動画制作
- App Store Connect 申請
